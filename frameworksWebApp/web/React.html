Framework React

Definicion

React es un framework para el desarrollo de aplicaciones web que utiliza como lenguaje principal JavaScript. Al igual que los otros framewors analizados, este se basa también en componentes (como un carrito de compras o la caja de login), pudiendo acceder a ellos por id y cambiar alguna propiedad. Este framework no es completo y requiere el uso de librerías externas para funciones más avanzadas.

Cuando y como se creo


React fue creada por Jordan Walke, en la empresa desarrolladora Facebook en el año 2011. Este framework, en comparación con otros vistos no se desarrolla o apoya sobre otro framework existente.
Hoy en día se hay muchas aplicaciones desarrolladas con React, siendo las más importantes los productos de la empresa Facebook, como pueden ser Whatsapp, Instagram, Discord...

Caracteristicas principales


- Es un framework multiplatarforma, teniendo soporte en todas.
- Está basado en los estándares web.
- Tiene acceso al framework sin conexión.
- Famoso por su estructura modular y su escalabilidad.
- Aunque de por sí no es el framework más seguro, tiene APIs que complementan las aplicaciones desarolladas en React frente a vulnerabilidades comunes.

¿Que ventajas tiene frente al resto?

Es un framework extendido mundial y ampliamente utilizado por muchas empresas desarrolladoras web.

React es conocido por ser uno de los frameworks más fáciles de aprender debido a su flexibilidad y su modularidad.

React tiene de todos los frameworks para desarrollo web la comunidad de librerías de todas las librerías JavaScript.

A diferencia con los otros frameworks que se muestran en esta web, es el único framework que es orientado a objetos.

React puede ser incorporados a otros frameworks con facilidad lo que le aporta una flexibilidad máxima.

Es conocido como el framework más rápido de todos debido a los algoritmo de reducción de operaciones que proporciona.
