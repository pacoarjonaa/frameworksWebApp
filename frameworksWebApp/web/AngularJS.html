Framework AngularJS

Definición

AngularJS es un framework de desarrollo web de código abierto en JavaScript que simplifica y facilita el desarrollo delo front-end.
Este framework funciona leyendo un código HTML con atributos concretos, los cualesa interpreta como directivas ng que se representan a tiempo real usando dichos atributos.

Cuándo y cómo se creó


AngularJS fue creador por Miško Hevery para la empresa desarrolladora Google en el año 2009; desde entonces ha sido la herramienta de desarrollo para grandes productos conocidos a nivel mundial como PayPal, Microsoft Team, Gmail...

Características principales


- Es un framework multiplatarforma, teniendo soporte en todas.
- Está basado en los estándares web.
- Tiene acceso sin conexión mediante la caché.
- Tiene una curva de aprendizaje elevada , haciendo que sea un poco difícil de empezar a usar.
- Incorpora muchas funciones de seguridad como codificación sensible al contexto y protección CSRF.

¿Que ventajas tiene frente al resto?

Es uno de los frameworks más potentes que existen hoy en día gracias a las múltiples utilidades que presenta.

Es el framework más extendido a nivel laboral.

Es muy flexible y proporciona soporte para sistemas sin oponer restricciones en la estructura del proyecto.
